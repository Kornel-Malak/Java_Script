1) Kiedy powstał Java script?

JavaScript został stworzony 4 grudnia 1995 roku.

2) Dlaczego jest to język skryptowy?

Nie musi zostać skompilowany do kodu maszynowego, aby można było zobaczyć efekty jego działania.

3) Gdzie jest wykonywany JS

Wykonuje się po stronie użytkownika strony internetowej.

4) Dlaczego jest JS o bezpieczne narzędzie

JavaScript ma znacznie ograniczone uprawnienia dostępu do zasobów komputera.

5) Czy jest to język obiekowy, jeśli tak to dlaczgo

Javascript jest językiem programowania obiektowego ponieważ wykorzystuje definicje obiektów, metod, zdarzeń, które mogą być zastosowane do opisu właściwości obiektów.

6) Wymień zalety JS:

1) Największą zaletą JavaScriptu jest niewątpliwie jego prostota.
2) Jest to obecnie najbardziej rozpowszechniony język skryptowy
3) Pomaga użytkownikom tworzyć nowoczesne aplikacje internetowe do bezpośredniej interakcji bez ponownego ładowania strony za każdym razem.

7) Framework to:

Rodzaj programu do tworzenia innych programów w JS.

8) Do aplikacji mobilnych można użyć Framework

Bardzo popularnym frameworkiem do tworzenia aplikacji jest React Native.

9)Aplikacje responsywne to:

JavaScript udostępnia metody, których można użyć w aplikacji, aby uzyskać bieżący rozmiar ekranu i wykorzystać wynik do wykonania działań w oparciu o ten rozmiar.

10) &lsaquo input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')" &rsaquo

input type="button" – Tworzy przycisk w formularzu.
name="przycisk" – Ustala nazwę przycisku (może być używane w skryptach).
value="Nowa Strona" – Tekst widoczny na przycisku.
onclick="WinOpen(' ')" – Wywołuje funkcję WinOpen() po kliknięciu przycisku. Funkcja ta ma otworzyć nowe okno.

11) window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

window.open() – Otwiera nowe okno przeglądarki z podanym adresem.
"obraz.html" – Strona, która zostanie otwarta.
"okienko" – Nazwa okna (używana wewnętrznie przez przeglądarkę).
toolbar=no – Ukrywa pasek narzędzi przeglądarki.
directories=no – Ukrywa pasek katalogów.
menubar=no – Ukrywa menu przeglądarki.
height=280 – Wysokość okna w pikselach.
width=160 – Szerokość okna w pikselach.
top=200 – Pozycja okna od górnej krawędzi ekranu w pikselach.
left=200 – Pozycja okna od lewej krawędzi ekranu w pikselach.

12) window.close()

window.close() – Zamyka okno.
